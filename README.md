# :construction_worker: How to run
**You need to install [Node.js](https://nodejs.org/en/download/), then:**

### Install Dependencies
```bash
npm install -g yarn
```

```bash
yarn
```


### Run Aplication

```bash 
yarn start 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
